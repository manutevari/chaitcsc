# CSC Services Data

This folder contains service-specific data, schemas, and sample documents for CSC schemes.

## Structure

- **csc/**: Common Service Centre guidelines and policies
- **pmkisan/**: PM-KISAN scheme documentation
- **passport/**: Passport service guidelines
- **ayushman/**: Ayushman Bharat health scheme data
- **digipay/**: DigiPay digital payment guidelines

## Usage

These data files are used by the Knowledge Base and agents for:
- Service discovery and recommendations
- Eligibility assessment
- Document verification
- Compliance checks
- Workflow guidance
